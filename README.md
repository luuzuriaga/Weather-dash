# 🌤️ Weather Dashboard

A beautiful, modern, and highly responsive weather dashboard application built with vanilla web technologies. It provides real-time weather information and forecasts with a stunning premium "glassmorphism" design.

## ✨ Features

- **Real-time Weather Data**: Get current temperature, "feels like" temperature, humidity, and wind speed.
- **Detailed Forecasts**: View the 24-hour weather forecast for any searched city.
- **City Search**: Quickly search for weather conditions across cities worldwide (e.g., London, Tokyo, Lima).
- **Save Favorites**: Bookmark your favorite cities for quick and easy access later.
- **Dynamic Backgrounds**: The application adapts visually to provide an immersive experience based on the searched location (like cityscapes).
- **Premium UI & Glassmorphism**: Features a sleek, modern interface with frosted glass effects and smooth hover animations.
- **Fully Responsive**: Optimized for all devices, from large desktop monitors to mobile screens, with a dedicated mobile slide-out sidebar navigation.

## 🛠️ Technologies Used

- **HTML5**: Semantic structure and layout.
- **CSS3**: Vanilla CSS for styling, responsive design (Media Queries), and modern visual effects (backdrop-filter for glassmorphism).
- **JavaScript (Vanilla)**: For robust DOM manipulation, API integration, and interactive UI logic.

## 🚀 Getting Started

### Prerequisites

You don't need any complex build tools to run this application. A modern web browser is all that is required.

### Installation & Execution

1. **Clone the repository** (if applicable):
   ```bash
   git clone <your-repository-url>
   ```
2. **Navigate to the project directory**:
   ```bash
   cd weather-dash
   ```
3. **Run the application local server**:
   You can use any local server, for example, Python's built-in `http.server`:
   ```bash
   python3 -m http.server 8080
   ```
4. **Open in Browser**:
   Navigate to `http://localhost:8080` in your preferred web browser.
